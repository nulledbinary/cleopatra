# Cleopatra

An application that is tailored for OSINTING People. With the help of databrokers and crawlers, everything is made possible!

This application is able to do the following:
- Pull SSS Registry of a person a data-leakage if possible.
- Get any deregatory records from Government Agency websites.
- Current Employer (If have one)
- Records of Pag-ibig and Philhealth whether the person is having a hardtime on his financial needs and things.
